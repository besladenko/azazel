# VK Posts and Likes Analysis

Это проект для анализа активности пользователей ВКонтакте, включая обработку постов и лайков.

## vk_posts

Модуль **`vk_posts`** предназначен для сбора ссылок на конкурсы со страницы которая занимается массовым участием.

---

## vk_likes

Модуль **`vk_likes`** анализирует лайки на постах ВКонтакте и вычисляет процент активности пользователей, основываясь на их лайках.

### Основные функции:
- Сбор лайков на заданных постах.
- Подсчет процента активности пользователей.
- Фильтрация пользователей по проценту лайков (например, более 75% лайков).

### Пример использования:
1. Введите ссылки на посты в текстовый файл.
2. Скрипт соберет все лайки для этих постов.
3. Получите список пользователей, поставивших лайк на 75% или более постов.
4. Результат будет записан в файл `lol.txt`.
